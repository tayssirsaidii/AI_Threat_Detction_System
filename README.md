This project is an AI-based cybersecurity service developed to detect cyber threats in enterprise environments using the CICIDS2017 dataset. It applies machine learning techniques to classify network traffic and identify potential attacks.

## Key Features

* Cyber threat detection using machine learning
* Random Forest Classifier for classification tasks
* Data preprocessing and feature engineering using the CICIDS2017 dataset
* Automated PDF report generation including:

  * Threat severity analysis
  * Security recommendations
  * Model performance evaluation

## Technologies Used

* Python
* Scikit-learn
* NumPy
* Pandas
* PostgreSQL
* PyCharm

## Project Objectives

* Detect and classify cyber threats in network traffic
* Improve enterprise security monitoring using machine learning
* Automate reporting for faster security response and analysis

## Outputs

* Threat classification results
* Model performance metrics (accuracy, precision, recall)
* Generated PDF reports summarizing security analysis
